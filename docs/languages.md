# Supported languages

Scribe transcribes with NVIDIA's Parakeet v3 model, running entirely on your
Mac's Neural Engine. It auto-detects which language you're speaking — there's
nothing to select before you start.

## The languages Scribe transcribes

Parakeet v3 covers 25 European languages:

- Bulgarian
- Croatian
- Czech
- Danish
- Dutch
- English
- Estonian
- Finnish
- French
- German
- Greek
- Hungarian
- Italian
- Latvian
- Lithuanian
- Maltese
- Polish
- Portuguese
- Romanian
- Russian
- Slovak
- Slovenian
- Spanish
- Swedish
- Ukrainian

## Languages Scribe does not yet cover

Anything outside the set above — including Arabic, Japanese, Mandarin, Hindi, and
Korean — is not supported yet. Scribe will still record audio if you speak an
unsupported language, but the transcript won't be accurate.

## English-only alternative

If you work only in English, Preferences also offers **Parakeet v2**, an
English-only model. You can switch to it instead of the multilingual v3 model.

> **Note:** Language detection and transcription both run on-device. No audio
> is uploaded, and no network connection is needed. See [Privacy](privacy.md).
